Persona 3 reload website

Frontend design

Bij deze case mochten we een website naar onze keuze coderen. Het leuke hieraan was dat je bij echt elk mini dingetje moest afvragen hoe de makers het hadden gecodeerd. Door afbeeldingen en fonts te downloaden van de originele website kon je echt een mooie kopie maken, en daarom ben ik ook erg trots op deze mobiele website.

github repository

Tijdens het namaken ontdek je veel leuke lay-outs, waar je waarschijnlijk zelf niet op was gekomen. Het goed krijgen van elke zin qua positie was een uitdaging waar ik veel plezier in ervaarde.

Een focus dit project was om je website te verbeteren ten opzichte van de originele. Dingen zoals semantisch coderen speelden daardoor een hoofdrol. Voordat ik begon met coderen heb ik de site eerst goed geanalyseerd, om zo tot een nette html structuur te komen. Deze manier van werken gaat denk ik ook handig zijn in toekomstige cases.

Een ander leerdoel in deze case was om veel te experimenteren met leuke functionaliteiten. Ik heb in deze case onder andere geleerd hoe je scrollanimaties doet. Maar het is me ook gelukt om bijvoorbeeld meerdere themes zoals een dark mode toe te voegen en nog veel meer dingen!
