Persona 3 reload website

Frontend design

Bij deze case mochten we een website naar keuze na maken. Het leuke hieraan was dat je bij elk onderdeel van de website moest afvragen hoe de makers het hadden gecodeerd. Hierdoor experimenteerde je veel meer dan normaal, waardoor ik denk dat ik bij dit project ook echt snel beter ben geworden in coderen.

github repository

Tijdens het namaken ontdekte je veel kleine dingen die veel moeilijker bleken te zijn dan je dacht. Ik ervaarde veel plezier in het oplossen van deze problemen.

Een focus dit project was om je website te verbeteren ten opzichte van de originele. Semantisch coderen speelde daardoor een hoofdrol. Voordat ik begon met coderen heb ik de site eerst goed geanalyseerd, om zo tot een nette html structuur te komen. Deze manier van werken gaat denk ik ook handig zijn in toekomstige cases.

Een ander leerdoel in deze case was om veel te experimenteren met leuke functionaliteiten. Ik heb in deze case onder andere geleerd hoe je scrollanimaties, themes en nog veel meer maakt!
